> # WebReceiver

> Requirements
```bash
# redis
go get github.com/go-redis/redis
# receiver
go get github.com/YuranIgnatenko/WebService
```
***

> Install
```
go get github.com/YuranIgnatenko/WebReceiver
```
***

> Use
```bash
cd ../WebReceiver
go run main.go
```

***

> Run test
```bash
cd receiver
go test receiver_test.go
```
